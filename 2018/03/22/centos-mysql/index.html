centos 安装mysql

Posted on 2018-03-22 Edited on 2024-03-21

注：该文档只适用centos7以下（不包括centos7），安装mysql5.6。文档中的mysql5.6相对于mysql5.7存在一些意想不到的问题。比如int类型的获取到undefined会被mysql处理成default值。

检测系统是否安装mysql

yum list installed | grep mysql

安装mysql安装包

查看可安装的mysql列表

yum list|grep mysql选择需要安装，一般只需要安装client和server即可。

启动mysql

service mysqld start
